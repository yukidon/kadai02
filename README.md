# 課題1 -じゃんけんアプリ　リッチVer-
課題2回目

## ①課題内容（どんな作品か）
- バーコードを読み取って作ったカードで、対戦をするゲームです

## ②工夫した点・こだわった点
- 天気によって敵の強さが変わる点
- カメラでバーコードを読み取り、文字列から味方のカードを生成する点
- ＜リッチ化＞
- 数の強さで勝ち負けを決めるのではなく、カードごとにHP、攻撃力、防御力を設定し勝負する形に変更
- ダメージはランダム
